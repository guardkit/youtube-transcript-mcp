# Development

Guide for contributors and developers working on YouTube Transcript MCP.

## Quick Start

```bash
git clone https://github.com/appmilla/youtube-transcript-mcp.git
cd youtube-transcript-mcp
pip install -e ".[dev]"
pytest tests/ -v
```
