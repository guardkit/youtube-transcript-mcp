# Getting Started

Set up YouTube Transcript MCP and fetch your first transcript in minutes.

## Setup Steps

1. **[Installation](installation.md)** - Install from source, verify prerequisites
2. **[Quick Start](quickstart.md)** - Fetch your first transcript in 5 minutes
3. **[Configuration](configuration.md)** - Claude Desktop integration, environment variables

## Prerequisites

- Python >= 3.10
- Claude Desktop (for MCP integration) or any MCP-compatible client

## Overview

YouTube Transcript MCP provides two ways to access its tools:

- **MCP Server** - Connects to Claude Desktop for conversational transcript access
- **CLI** - Direct terminal access for scripting and automation

Both interfaces provide the same capabilities: fetching transcripts, listing available languages, and extracting structured insights from video content.
